# TestFacePI

## 安裝 anaconda3 並勾選設定環境變數

## 建立隔離空間
    conda create -n facepi python=3.6.9
    conda activate facepi
## 安裝 相關套件
    pip install opencv-python Pillow mediapipe imutils

## 安裝 dlib
    pip install dlib
